Исполнитель оказывает Пользователю услугу по организации перевода добровольного пожертвования (донат) выбранному стримеру-партнёру, зарегистрированному на платформе StreamCanvas. По желанию Пользователь может сопроводить донат собственным изображением (рисунком), нарисованным на сайте сервиса.

4. Порядок оказания услуги и способ получения результата

Услуга оказывается полностью дистанционно, без физической доставки:

  1. Пользователь оплачивает донат через платёжный сервис ЮKassa.
  2. Услуга по организации перевода доната считается оказанной в момент успешного зачисления платежа Исполнителем.
  3. Если донат сопровождался изображением, оно направляется стримеру-партнёру в личный кабинет на модерацию.
  4. Стример-партнёр самостоятельно принимает решение о демонстрации изображения в эфире и определяет момент показа в пределах оплаченного времени.

5. Модерация изображения и возврат денежных средств

Изображение, приложенное к донату, проходит модерацию стримером-партнёром. Стример-партнёр вправе отклонить показ изображения по любой причине без объяснения. Решение стримера-партнёра окончательное.

Основная ценность услуги — организация перевода пожертвования стримеру-партнёру; демонстрация изображения — сопутствующая опция. В связи с этим денежные средства за донат не возвращаются в случае отклонения изображения модерацией, за исключением случаев, предусмотренных разделом 6.
